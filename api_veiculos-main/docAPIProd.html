Documentação da API REST - Veículos

1. Listar Todos os Veículos

Método: GET

Rota: /veiculos

Exemplo de resposta:
[ 
  { 
    "id": 1, 
    "marca": "Ford", 
    "modelo": "Fiesta", 
    "ano": 2015, 
    "preco": 30000.00 
  } 
]

2. Listar um Veículo Específico

Método: GET

Rota: /veiculos/<int:veiculo_id>

Exemplo de resposta:
{ 
  "id": 1, 
  "marca": "Ford", 
  "modelo": "Fiesta", 
  "ano": 2015, 
  "preco": 30000.00 
}

3. Criar um Novo Veículo

Método: POST

Rota: /veiculos

3. Criar um Veículo

Método: POST

Rota: /veiculos

    Corpo da requisição (JSON):
    {
      "marca": "Ford",
      "modelo": "F-150 Raptor",
      "ano": 2022,
      "preco": 470000.00
    }
    
    Resposta:
    {
      "id": 2,
      "marca": "Ford",
      "modelo": "F-150 Raptor",
      "ano": 2022,
      "preco": 470000.00
    }

4. Atualizar um Veículo

Método: PUT

Rota: /veiculos/<int:veiculo_id>

    Corpo da requisição (JSON):
    {
      "marca": "Tesla",
      "modelo": "Model Y",
      "ano": 2023,
      "preco": 380000.00
    }
    
    Resposta:
    {
      "id": 1,
      "marca": "Tesla",
      "modelo": "Model Y",
      "ano": 2023,
      "preco": 380000.00
    }

5. Deletar um Veículo

Método: DELETE

Rota: /veiculos/<int:veiculo_id>

    Resposta:
    {
      "mensagem": "Veículo excluído com sucesso"
    }
